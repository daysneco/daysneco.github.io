## 今日最重要的7条 AI 新闻

### 1. 苹果起诉 OpenAI 窃取商业秘密，昔日合作伙伴彻底翻脸
- 来源：ABC News / Fortune
- 链接：https://abcnews.com/Technology/wireStory/apple-files-lawsuit-accusing-chatgpt-maker-openai-stealing-134663221
- 这件事是什么：7 月 10 日，苹果在加州联邦法院起诉 OpenAI，指控其通过前员工系统性窃取苹果商业秘密，用于开发 ChatGPT 硬件设备。被告包括 OpenAI 硬件负责人 Tang Tan（曾参与 iPhone/Apple Watch 设计）和前苹果工程师 Chang Liu。诉状还点名 Jony Ive 创立的 io Products（已被 OpenAI 收购）。
- 为什么重要：这是苹果与 OpenAI 合作关系的重大破裂。两家 2024 年刚把 ChatGPT 接入 Siri，如今却因 OpenAI 做硬件对簿公堂。苹果指控 Tan 离职前把供应商信息和内部研发资料发给自己邮箱，还鼓励同事带走零件。
- 对我有什么影响：短期内你用 iPhone 上的 ChatGPT 功能不会立刻变，但长期可能影响苹果和 OpenAI 的合作深度。如果你在关注 AI 硬件赛道，这场官司会直接影响 OpenAI 首款设备的上市节奏和供应链布局。

### 2. GPT-5.6 Sol 独立评测出炉：编程第一，成本只有 Fable 5 的三分之一
- 来源：Artificial Analysis / The Decoder
- 链接：https://artificialanalysis.ai/articles/gpt-5-6-has-landed
- 这件事是什么：7 月 10 日，独立评测机构 Artificial Analysis 发布 GPT-5.6 全系测评。Sol 在综合智能指数得 59 分，仅比 Claude Fable 5 低 1 分，但在编程 Agent 指数以 80 分夺冠。更关键的是，Sol 每任务成本约 1.04 美元，约为 Fable 5（2.75 美元）的三分之一；Terra 和 Luna 更便宜。
- 为什么重要：这是 GPT-5.6 发布后的第一份权威第三方验证，坐实了 OpenAI「用更低价格打 Anthropic」的策略。GPT-5.6 还首次引入 cache-write 定价，和 Anthropic 看齐，长期用 API 的成本结构会变化。
- 对我有什么影响：如果你在选编程 AI 或企业 API，现在有了清晰的性价比参考：要最强编程能力选 Sol，要省钱试 Terra（77 分）或 Luna（75 分）。建议拿你自己的真实任务跑一遍，别只看榜单分数。

### 3. ChatGPT Chrome 扩展升级：侧边栏直接读网页，正面迎战 Google Gemini
- 来源：Engadget / Digital Trends
- 链接：https://www.engadget.com/2212701/openai-browser-isnt-dead/
- 这件事是什么：7 月 10 日，多家科技媒体跟进报道 OpenAI 更新的 ChatGPT Chrome 扩展。扩展运行在 Chrome 侧边栏，能读取你正在浏览的网页内容，帮你总结文章、回答问题、甚至启动多步 Agent 任务，无需复制粘贴。这和 Google Chrome 内置的 Gemini 侧边栏功能直接对标。
- 为什么重要：OpenAI 关停 Atlas 浏览器后，把「浏览器 AI」能力拆成两块：桌面版内置浏览器 + Chrome 扩展。这意味着 OpenAI 和 Google 在浏览器入口上的正面战争正式开始，谁控制了「你看网页时的 AI」，谁就掌握了大量日常办公场景。
- 对我有什么影响：Chrome 用户可以在扩展商店安装 ChatGPT 扩展试用。注意授权范围——它需要你允许读取当前页面内容。如果你处理敏感内部文档，先想清楚隐私边界再开权限。

### 4. MiniMax 限售股解禁，两天暴跌 27%，市值蒸发超 200 亿港元
- 来源：搜狐财经 / 观察者网
- 链接：https://www.sohu.com/a/1048498824_115479
- 这件事是什么：7 月 9 日 MiniMax 迎来上市后首轮限售股解禁，股价单日暴跌 17.98%，10 日再跌约 9%，两天累计跌幅约 27%，市值从约 932 亿港元跌至约 850 亿港元。花旗 7 月 6 日已将目标价从 1330 港元大幅下调至 533 港元，理由是 M3 模型市场反应偏淡、用户留存和变现策略不确定。
- 为什么重要：这是港股 AI 概念股波动加剧的信号。MiniMax 股东包括腾讯、高瓴、真格等，财务型投资者占解禁盘超三分之一，获利了结压力大。与此同时，公司还在推进 2.7 万亿参数新模型的研发，资本市场和基本面出现明显分歧。
- 对我有什么影响：普通 AI 用户短期无感，但如果你持有港股 AI 概念股或在评估 MiniMax 的 API/产品，需要关注：股价波动不等于产品不行，但说明市场对「大模型公司怎么赚钱」越来越挑剔。

### 5. MiniMax 计划推出 2.7 万亿参数 M3 Pro，三季度开源
- 来源：The Information / 腾讯新闻
- 链接：https://news.qq.com/rain/a/20260708A088FS00
- 这件事是什么：7 月 8-10 日，The Information 报道 MiniMax 正在研发代号 M3 Pro 的新一代大模型，参数量达 2.7 万亿，计划最早 2026 年三季度发布并开源。这将是目前中国公司计划发布的最大规模开源模型，重点提升复杂逻辑推导和多步骤任务处理能力。
- 为什么重要：中国开源模型军备竞赛又升一级。DeepSeek、智谱、月之暗面都在推大参数开源模型，MiniMax 如果兑现，会给全球开发者提供一个免费且极强的选项，进一步挤压美国闭源模型的定价空间。
- 对我有什么影响：三季度前可以先观望，等权重发布再实测。如果你现在就需要便宜好用的开源模型，GLM-5.2、DeepSeek-V4 已经够用，不必等 M3 Pro。

### 6. Google Gemini 3.5 Pro 据传 7 月 17 日发布，DeepSeek API 7 月 24 日迁移截止
- 来源：TechTimes / Business Insider
- 链接：https://www.techtimes.com/articles/319877/20260708/gemini-35-pro-targets-july-17-deepseeks-july-24-deadline-hits-developers-now.htm
- 这件事是什么：多家媒体报道 Google 目标在 7 月 17 日发布 Gemini 3.5 Pro（Google 尚未官方确认），据称 Google 放弃了原 2.5 Pro 底座、重新预训练以追赶 GPT-5.6 和 Fable 5。同一周，DeepSeek 宣布 7 月 24 日起旧版 API 别名 deepseek-chat 和 deepseek-reasoner 将停止响应，开发者必须迁移到 V4 系列。
- 为什么重要：下周可能是今年 AI 模型发布最密集的一周：Gemini 3.5 Pro、DeepSeek V4 正式版、以及各界都在等的更多动作。对开发者来说，DeepSeek 的 7 月 24 日截止日是硬期限，不迁移服务会中断。
- 对我有什么影响：用 DeepSeek API 的开发者，这周内检查代码里的模型名称，尽快切到 V4-Pro 或 V4-Flash。关注 Google 发布的朋友，7 月 17 日前后留意官方公告，但目前仍是「据传」，别提前把生产环境押上去。

### 7. GPT-5.6 新增 Programmatic Tool Calling：模型自己写程序调工具，减少无效往返
- 来源：OpenAI 官方 / Meyka
- 链接：https://openai.com/index/gpt-5-6/
- 这件事是什么：GPT-5.6 在 API 层面推出了 Programmatic Tool Calling（程序化工具调用）。模型可以编写并运行轻量级程序来协调多个工具、处理中间结果，只把有用信息传回上下文，避免传统 tool call 一来一回的 token 浪费。同时支持显式 cache 断点、30 分钟最低缓存寿命和 90% 缓存读取折扣。
- 为什么重要：这是 Agent 开发效率的实质性升级。做复杂自动化工作流时，token 消耗和延迟往往是最大成本，程序化调用能显著降低两者。对 API 开发者来说，GPT-5.6 的定价体系（含 cache-write 费用）也需要重新算账。
- 对我有什么影响：如果你在写 AI Agent 或自动化流水线，值得在 GPT-5.6 API 上试试 Programmatic Tool Calling，特别是需要串联 3 个以上工具的场景。先在测试环境对比传统 tool call 的 token 消耗和完成时间，再决定是否切换生产。
